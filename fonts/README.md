# Ion Font Library

标准字形库：

  - sculpture铭文体：用利器刻写在木石材料上，近乎原始的文字、现在只用于外来语转写
  - scroll卷轴体：用墨水写在较柔软的皮纸材料上，用于圣书和公文(推荐)
  - handwriting手写体：卷轴体的草书简化体，现代文书

自定义字形库方法：

  0. 在本目录(fonts)下以字形库名新建文件夹，如`newal`
  1. 字库名中每个文字为独立的一个png文件(背景透明或留白都可)，文件名为文字对应的**拉丁文转写**，如`k.png`
